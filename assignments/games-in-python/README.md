# 📘 Assignment: Hangman Game in Python

## 🎯 Objective

Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts. You'll practice string manipulation, loops, conditionals, and random selection.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description

Build the classic word-guessing game using Python strings, loops, and user input. Your game should randomly select a word, allow the player to guess letters, display progress, track attempts, and determine if the player wins or loses.

#### Requirements

Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and show current progress (_ _ _ format)
- Track incorrect guesses remaining
- End when the word is guessed or attempts are exhausted
- Display win/lose messages based on the game outcome
